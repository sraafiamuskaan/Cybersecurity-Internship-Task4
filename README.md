# Task 4 – Exploitation and System Security

## ApexPlanet Software Pvt. Ltd. – Cyber Security Internship

### Overview

This repository contains the work completed for **Task 4 – Exploitation and System Security** as part of the ApexPlanet Cyber Security Internship.

The objective of this task was to understand the penetration testing process by performing controlled exploitation, password auditing, malware analysis, and implementing basic system hardening techniques in a safe laboratory environment.

---

## Objectives

- Understand the penetration testing methodology.
- Perform exploitation using the Metasploit Framework.
- Learn password auditing using Hydra and John the Ripper.
- Study malware analysis concepts.
- Implement basic Linux system hardening techniques.
- Document findings and recommended security practices.

---

## Tools Used

- Kali Linux
- Metasploit Framework
- Hydra
- John the Ripper
- OpenSSL
- iptables
- Linux Terminal
- SHA-256 Utilities

## Activities Performed

### 1. Penetration Testing Methodology
- Information Gathering
- Vulnerability Assessment
- Exploitation
- Reporting

### 2. Exploitation
- VSFTPD Backdoor
- UnrealIRCd Backdoor
- DistCC Remote Command Execution

### 3. Password Auditing
- SSH password attack using Hydra
- Password hash cracking using John the Ripper

### 4. Malware Analysis
- File type identification
- SHA-256 hash generation
- Static and Dynamic Analysis concepts

### 5. System Hardening
- Firewall inspection
- Running service analysis
- Open port inspection
- Package updates
- Security best practices
